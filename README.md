# viruchki

# Сайт для выручки ресторанов

## Что умеет
- Загружает один или несколько Excel-файлов (`.xlsx`).
- Ищет дату, ресторан и сумму выручки в типовых отчетах.
- Агрегирует выручку по схеме: `дата + ресторан`.
- Показывает:
  - общую выручку,
  - количество ресторанов,
  - количество дней,
  - график по дням,
  - таблицу с фильтрами по ресторану и периоду.

## Как открыть
1. Откройте файл `index.html` в браузере.
2. Нажмите выбор файлов и загрузите ваши Excel-отчеты.
3. Используйте фильтры для анализа.

## Важно
- Для чтения `.xlsx` используется библиотека SheetJS через CDN.
- Нужен доступ к интернету в момент открытия страницы, чтобы подгрузился скрипт библиотеки.
